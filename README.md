# Analog CMOS Labs

This repository contains laboratory reports for the **Analog CMOS Circuits** course.

## Included Labs

### Lab 1 – Source Follower and Cascode
- Source follower biasing and DC behavior
- Bulk effect analysis
- Cascode current mirror
- Output resistance (Rout) evaluation
- DC and AC simulations using Cadence Virtuoso

### Lab 2 – Differential Amplifier
- Differential to differential amplifier design
- Common-mode analysis
- Differential gain (Av)
- Small-signal parameters (gm, Rout)
- Current steering and headroom analysis

  ### Lab 3 – 5T Operational Amplifier with Source Follower
– Differential-to-single-ended 5T op-amp design  
– Unity-gain buffer (UGB) configuration  
– Headroom and saturation verification  
– Small-signal gain analysis (Av, gm, Rout)  
– Transient step response and settling time  
– AC analysis and Bode plot  
– Phase margin evaluation and compensation
